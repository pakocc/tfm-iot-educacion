# Anexo C. Configuración del broker MQTT (Mosquitto)

Este anexo recoge la configuración completa del broker MQTT utilizado en la infraestructura IoT. Se incluyen los archivos de configuración del servicio Mosquitto, la estructura de directorios, la definición de usuarios y contraseñas, y las listas de control de acceso (ACL) que garantizan el aislamiento entre los estudiantes.

El objetivo es proporcionar una guía exhaustiva que permita reproducir el entorno de mensajería de forma segura y estable.
