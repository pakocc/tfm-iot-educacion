# E.2. Bibliotecas utilizadas

El firmware se desarrolló en el entorno Arduino IDE utilizando las siguientes bibliotecas:

- **[ESP8266WiFi.h](ca://s?q=Biblioteca_ESP8266WiFi)**: gestión de la conexión Wi‑Fi.
- **[PubSubClient.h](ca://s?q=Biblioteca_PubSubClient)**: implementación del cliente MQTT.
- **[DHT.h](ca://s?q=Biblioteca_DHT11)**: lectura del sensor DHT11.
- **[Servo.h](ca://s?q=Biblioteca_Servo_Arduino)**: control del microservo SG90.

Estas bibliotecas permiten una comunicación eficiente y estable con el broker MQTT y la integración de sensores y actuadores en la infraestructura IoT.
