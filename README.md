# Infraestructura IoT para entornos educativos — Material complementario del TFM

Este repositorio contiene el material técnico completo asociado al Trabajo Fin de Máster "Despliegue de una infraestructura IoT multiusuario para entornos educativos".

Debido al límite de extensión del documento principal, los anexos completos se incluyen aquí como material complementario.

## Contenido del repositorio

- **docs/**: Documentación técnica ampliada (arquitectura, instalación, despliegue, pruebas).
- **nodered/**: Flujos completos de Node-RED para cada usuario.
- **mosquitto/**: Configuración del broker MQTT (Mosquitto).
- **nodemcu/**: Código fuente de los microcontroladores (ESP8266/NodeMCU).
- **dashboards/**: Dashboards de Node-RED (capturas y exportaciones).
- **anexos/**: Anexos completos tal como se entregaron originalmente.

## Reproducibilidad

El repositorio incluye:
- Archivos de configuración
- Dockerfiles
- Flujos de Node-RED
- Código de dispositivos IoT
- Diagramas técnicos
- Guías paso a paso

Esto permite replicar la infraestructura en cualquier Raspberry Pi compatible.
