# F.8. Resumen de la arquitectura de red

La arquitectura híbrida implementada permite:

- [garantizar accesibilidad tanto local como remota](ca://s?q=Accesibilidad_local_remota_IoT),
- [mantener un entorno seguro mediante Tailscale](ca://s?q=Seguridad_Tailscale_IoT),
- [asegurar baja latencia para la comunicación IoT](ca://s?q=Baja_latencia_IoT),
- [facilitar la supervisión docente desde cualquier ubicación](ca://s?q=Supervision_docente_remota),
- [integrar dispositivos IoT reales en un entorno controlado](ca://s?q=Integracion_dispositivos_IoT).

Esta arquitectura constituye un componente esencial de la infraestructura IoT, proporcionando la base de conectividad necesaria para el funcionamiento coordinado de contenedores, dashboards y dispositivos físicos.
