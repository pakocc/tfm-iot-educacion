# Anexo B. Despliegue de contenedores Docker

Este anexo documenta el proceso completo de despliegue de los contenedores Docker que conforman la infraestructura multiusuario desarrollada. Se incluyen los archivos de configuración utilizados, la estructura de volúmenes persistentes, la definición de redes internas y los comandos necesarios para la creación y gestión de los contenedores.

El objetivo es garantizar la reproducibilidad del sistema y facilitar su implementación en otros entornos educativos.
